# politdata — партійні фінанси НАЗК у зручних таблицях

**Останнє оновлення: 2026-07-20 09:50**

Готовий до аналізу набір таблиць про фінанси політичних партій України. Дані беруться з
офіційного публічного API ПОЛІТДАТИ (НАЗК) `https://politdata.nazk.gov.ua/api/v2`, автоматично
чистяться й збагачуються, і щотижня оновлюються.

### Про відсутні таблиці
- **9.4 (повернення помилкових внесків)** — у новому API немає окремого джерела для цієї таблиці,
  тож вона не формується (повернення відображені всередині таблиць 9.3 та 9.5).
- **0_report_duplcates (дублікати звітів)** — зʼявляється лише коли в даних дійсно є задубльовані
  звіти. Новий API позначає видалені/непідписані звіти, тож зазвичай дублікатів немає й файлу теж.
- **3.4 (цінні папери)** — файл є, але наразі порожній: у жодному звіті нового API немає цих даних.

## Як це оновлюється

Раз на тиждень (понеділок, ранок) робот на серверах GitHub сам качає свіжі звіти з API,
перебудовує всі таблиці й публікує їх у розділ **Releases** цього репозиторію. Кнопки завантаження вище завжди ведуть на останній випуск.
